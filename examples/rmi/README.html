Overview

  • HelloWorld

    This example is the usual basic example using Vortex OpenSplice RMI.

  • ClientServer

    This example demonstrates RMI invocations of a more advanced service interface having different IDL operation forms with different parameter types and passing modes.

  • Printer

    This example is an advanced example using synchronous and asynchronous invocation modes of OpenSplie RMI.

Each example is written in Java and C++.

Building the examples

The Vortex OpenSplice RMI examples are located in $OSPL_HOME/examples/rmi directory. Each example has a Java and a C++ version. The following sections explain the build procedure.

Environment Setup

Let's call OpenSplice_install_dir the Vortex OpenSplice installation directory.

The Vortex OpenSplice environment variables must be set to build and run the examples correctly.

On Linux, open a terminal and source the "OpenSplice_install_dir/release.com" script supplied with the distribution.

On Windows, The Vortex OpenSplice Command Prompt can be selected from the start menu. Alternatively, open a windows command prompt and execute the "OpenSplice_install_dir\release.bat" batch script supplied with the distribution.

Linux build

C++ examples

The C++ examples can be built using the provided makefiles either individually from the example-specific sub-directory, or jointly from $OSPL_HOME/examples/rmi global directory.

To build all the RMI C++ examples, please type these commands :

    $ cd $OSPL_HOME/examples/rmi
    $ make

Java examples

Each Java example can be built from the java sub-directory of the example directory. The following commands show how to build the HelloWorld Java example on Linux :

    $ cd $OSPL_HOME/examples/rmi/HelloWorld/java
    $ make

Windows build

C++ examples

The C++ examples can be built in Visual Studio using the provided solution files. After installing Vortex OpenSplice, from the start menu select the installed version of Vortrex OpenSplice, then select Examples/Standalone C and C++ Examples. This item will build all C and C++ examples including the RMI ones.

Java examples

A set of Windows batch files are provided to build the Java examples on Windows. These files are located in the java directory of each example-specific sub-directory. Start an Vortex OpenSplice command prompt and simply run the BUILD.bat script. 

C:\Program Files\PrismTech\OpenSpliceDDS\V6.2.0\HDE\x86.win32>cd examples\rmi\HelloWorld\java
C:\Program Files\PrismTech\OpenSpliceDDS\V6.2.0\HDE\x86.win32\examples\rmi\HelloWorld\java>BUILD.bat
